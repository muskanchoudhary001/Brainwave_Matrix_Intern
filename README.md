# 🌙 MOONEX - Landing Page  

This repository contains **Task 1** of my internship at **Brainwave Matrix Solutions**. As part of my internship, I developed a **fully responsive landing page** for a website named **MOONEX**, ensuring an appealing UI/UX.  

## 🚀 Project Overview  

MOONEX is a modern and sleek landing page designed to showcase the features and services of a website. The page is built using **React.js with JavaScript** and follows best practices for responsiveness and user experience.  

## 🛠 Technologies Used

## 🛠 How to Run Locally  
1. Clone the repository:  
   ```bash
   git clone https://github.com/muskanchoudhary001/Brainwave_Matrix_Intern.git
   ```
2. Navigate to the project folder:  
   ```bash
   cd moonex-landing-page
   ```
3. Install dependencies:  
   ```bash
   npm install  # or yarn install
   ```
4. Start the development server:  
   ```bash
   npm start  # or yarn start
   ```
5. Open `http://localhost:5173` in your browser to view the landing page.
